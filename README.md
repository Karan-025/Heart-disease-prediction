# Heart-disease-prediction
# Heart Disease Prediction System (Part 3)

## Overview

This project predicts the likelihood of heart disease using Machine Learning.

This is an improved version (Part 3) where 4 model are used to compare best out of them

## Features

* User-friendly prediction interface
* Trained ML model with improved accuracy
* Handles real-world medical input parameters

---

## Problem Statement

Heart disease is one of the leading causes of death worldwide.
The goal is to predict whether a person is at risk based on medical attributes.

---

## 📊 Dataset

* Source: UCI Heart Disease Dataset
* Dataset is included in the `Dataset/` folder
* Features include:

  * Age
  * Sex
  * Chest Pain Type
  * Cholesterol
  * Max Heart Rate
  * etc.
